# NTUST Thesis Typst Template

臺灣科技大學論文模板。Thesis template for National Taiwan University of Science and Technology.

## Usage

Create a new project using this template:

```shell
typst init @preview/ntust-thesis:0.1.0
```

And then compile:

```shell
typst compile main.typ
```

### Fonts

預設/建議使用[教育部標準楷書](https://language.moe.gov.tw/001/Upload/Files/site_content/M0001/edukai-5.0.zip) (TW-MOE-Std-Kai)
